# DesafioGlobo - Encurtador de URLs

**Instruções de setup**

Para fazer o setup, faça realize os seguintes comandos:

```
mkdir /opt/globo/
```
```
cd /opt/globo/
```
```
git clone https://github.com/SophiaDuboc/DesafioGlobo.git
```
```
cd DesafioGlobo/
```
```
make setup
```
```
make run
```

Feito, setup finalizado!
